### 项目简介

#### 软件功能

包含3个python程序

分别是：

traditionAlg/main.py：基于OpenCV自带定位算法kcf追踪算法实现多目标追踪

ssd_dlib/mian_ssd_dlib.py：单进程，使用dnn调用ssd模型识别目标，使用dlib进行目标追踪

ssd_dlib/main_ssd_dlib_multi.py：多进程，使用dnn调用ssd模型识别目标，使用dlib进行目标追踪
